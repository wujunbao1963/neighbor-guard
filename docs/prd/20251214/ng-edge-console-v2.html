NG Edge Console v2

Full implementation with all 7 tabs - Open in browser to view complete UI

Key Features Demonstrated:

  1. Alarm State Widget - Entry delay countdown with disarm options
  2. Mode Selector - HOME/AWAY/NIGHT/DISARMED
  3. 7 Navigation Tabs:
    • Overview - Dashboard with stats and recent events
    • Events - Full timeline with feedback
    • Devices - Camera health, battery, network monitoring
    • Topology - Visual zone map with signal animation
    • Contracts - Three-layer architecture (Signal/Evidence/Event)
    • Settings - Disarm channels, pre-intrusion tiers, timing
    • Neighbors - Collaboration actions and state machine
  4. Local Disarm Channels - Keypad/SmartLock/NFC options
  5. Pre-intrusion Tiers - Silent/Deterrence/Escalation
  6. Neighbor Actions - Acknowledge/Investigate/Record/Escalate
